# Advanced-Shell-Scripting
## Introduction

This project focuses on advanced shell scripting techniques used in Linux system administration and DevOps environments. The goal is to develop a fully automated system health monitoring script that checks key system resources such as CPU usage, memory usage, disk usage, and running processes.

In addition to system monitoring, the project demonstrates automation using cron jobs to schedule periodic execution of the script. It also includes an optional email alert system that notifies the administrator when system performance exceeds defined thresholds, such as high CPU usage.

By completing this project, learners gain practical experience in writing Bash scripts, automating administrative tasks, scheduling jobs in Linux, and implementing basic system alert mechanisms. These are essential skills for managing real-world server environments and building reliable DevOps workflows.
